## 종속성

## 환경
이 프로그램은 충분히 테스트되지 않았습니다. 실행 시 오류를 뿜을 수 있습니다. 주의해주세요<br>
이 어플리케이션은 Samsung Galaxy A16, Samsung Galaxy S24, Samsung Galaxy tab 등의 제한된 환경에서만 테스트되었습니다.
## 실행
실행 시 각종 종속성의 네임스페이스 오류가 뿜어져나올 수 있습니다. 주의하십시오.
